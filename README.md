# Pokemon Modern Emerald - Worped

This is a simple edit of Resetes12's Modern Emerald ROM-hack to include a few features I wanted. 

Includes:
* DexNav
* Added some Pokemon from later gens that I like (WIPs)
	- Lechonk + (Oinkologne)
	- Stufful + (Bewear)
	- (Mimikyu)
	- (Hatenna + Hattrem + Hatterene)
	- (Buneary + Lopunny)
* Move Info changed to L instead of Start
* Catch Mode

The code is really messey in some areas, this is my first time messing with this kind of thing.
